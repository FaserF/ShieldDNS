Desktop Setup (macOS & Windows)

macOS 14 (Sonoma) and later

1. Click the Download Config Profile button above.
2. Open System Settings → Privacy & Security → Profiles.
3. Double-click the downloaded ShieldDNS profile and click Install.
{{if not .SignEnabled}}Note: The "Not Signed" warning is normal for auto-generated profiles.{{end}}

Windows 11 (Native DoH)

1. Go to Settings → Network & internet → Wi-Fi (or Ethernet) → Hardware properties.
2. Next to "DNS server assignment", click Edit and change to Manual.
3. Enable IPv4. Set "Preferred DNS" to the IP address of this server.
4. Set "DNS over HTTPS" to On (manual template).
5. Under DoH template, enter:

https://{{.Host}}/dns-query

6. Click Save.
